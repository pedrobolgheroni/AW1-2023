Aula de AW1 04/07/2023

Atividade DOM

Descrição da atividade:

  1. Faça um site de sua escolha com no mínimo 35 elementos (tags), você deve usar elementos internos
  2. Você deve estilizar toda a página, buscando que todos os elementos tenham algum estilo.
  3. Montar um diagrama da árvore DOM, a partir do objeto “document”.
  4. Alterar 2 estilos de pelo menos 2 elementos de uma das 5 formas de acesso a elemento:
    • por Marca
    • por ID
    • por Nome
    • por Classe
    • por Seletor

Utilizei HTML, CSS e JS para fazer essa atividade
